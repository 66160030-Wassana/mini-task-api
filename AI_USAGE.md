# การใช้ AI (AI usage)
โปรเจกต์ Mini Task API นี้ได้รับความช่วยเหลือจาก AI (Artificial Intelligence) โดยเฉพาะโมเดลภาษา (Large Language Model) ในการช่วยเร่งกระบวนการพัฒนา ดังนี้
- สิ่งที่ AI ช่วย เช่น
  - ให้คำปรึกษาเกี่ยวกับโครงสร้างโปรเจกต์
  - สร้างโค้ดบางส่วน เช่น taskController ในส่วนที่ขาดหายไป
  - ช่วยแก้ไขข้อบกพร่องของโค้ด

- สิ่งที่ทีมทําเองทั้งหมด เช่น
  - การใช้ AI Prompt
  - การคิดวิเคราะห์ระบบ และฟังก์ชั่น
  - การปรับแต่งโค้ดให้ทำงานได้จริง
  - สร้าง Test Cases ใน Postman

- AI ถูกใช้เป็น เครื่องมือช่วยพัฒนา (development assistant) เท่านั้น ไม่ได้ใช้สร้างโปรเจกต์อัตโนมัติทั้งหมด
- ทีมผู้พัฒนาเข้าใจโค้ดทุกส่วน และสามารถอธิบายการทํางานได้ทั้งหมด
- โมเดล AI ที่ใช้คือ Generative AI (Google Gemini, ChatGPT)
